PlantScanner

PlantScanner is an innovative application designed to help users identify plants through image recognition. By simply scanning a photo of a plant, users can instantly receive detailed information about the plant species, including its name, characteristics, and care instructions. The app leverages the power of Azure Cognitive Services for advanced image processing and plant identification, providing accurate and reliable results. The project is built with a robust tech stack that includes React for a seamless and interactive web interface and React Native for a smooth and responsive mobile experience. The backend is powered by Node.js with GraphQL for efficient data querying, allowing for fast and flexible API interactions. Data management is handled using MongoDB, ensuring scalable and secure storage for user data and plant information. Together, these technologies create a powerful and user-friendly platform that allows plant enthusiasts, gardeners, and curious users to explore and learn more about the natural world around them.
